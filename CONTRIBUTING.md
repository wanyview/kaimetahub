# CONTRIBUTING

感谢您对 KaiMetaHub 的关注！

## 如何贡献

### 提交文章

1. Fork 本仓库
2. 创建分支: `git checkout -b article/your-topic`
3. 在 `articles/` 目录创建新文件
4. 提交 Pull Request

### 文章格式要求

```markdown
# 标题

## 引言
（简短介绍）

## 第一部分
...

## 第二部分
...

## 结语
...
```

### 提交播客

1. 在 `podcasts/` 目录创建新文件
2. 包含对话要点和精华摘录
3. 提交 Pull Request

### 提交问题

欢迎提交 Issue：
- 内容建议
- 错误指正
- 功能请求

## 联系方式

- GitHub Issues
- 提交 Pull Request
