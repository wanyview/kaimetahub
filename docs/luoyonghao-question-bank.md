## 附录：问题使用指南

### 采访流程建议

```
开场（5分钟）: Q1, Q2, Q3 → 建立信任
背景（15分钟）: Q6, Q7, Q8, Q9, Q10 → 了解经历
深入（30分钟）: Q21-Q50 → 事业探讨
碰撞（15分钟）: Q51-Q80 → 观点交流
升华（5分钟）: Q91-Q100 → 人生总结
```

### 追问技巧

| 触发词 | 使用时机 | 示例 |
|--------|---------|------|
| "刚才说的" | 需要深入 | "你刚才说的XX，能再展开讲讲吗？" |
| "为什么" | 挖掘原因 | "你为什么这么认为？" |
| "比如" | 要例子 | "能给个例子吗？" |
| "如果" | 做假设 | "如果再来一次，你会怎么选？" |
| "有没有" | 扩展范围 | "有没有什么时候觉得自己错了？" |
