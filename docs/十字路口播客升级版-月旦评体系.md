# 罗永浩十字路口播客升级版 - 月旦评体系

> 基于月旦评理念的AI采访与观点评选系统

---

## 一、系统定位升级

### 1.1 核心理念
| 要素 | 说明 |
|------|------|
| 评 | 客观评价每位嘉宾的观点与价值 |
| 论 | 深度讨论，挖掘背后逻辑 |
| 辩 | 观点碰撞，激发思辨 |
| 选 | 精选最有价值内容，形成榜单 |

### 1.2 差异化特色
- **周期性**：每周评选，持续更新
- **客观性**：AI作为第三方，不偏不倚
- **深度性**：不只是采访，更有深度解读
- **实用性**：指导实践，解决实际问题

---

## 二、栏目规划

### 📊 【周榜】本周观点TOP10
每周从采访内容中筛选10个最有价值的观点

**评分标准**：
| 维度 | 权重 |
|------|------|
| 创新性 | 25% |
| 深度 | 25% |
| 实用性 | 25% |
| 传播力 | 25% |

---

### 🎯 【月旦】月度人物评选
每月评选1位"月度人物"

**评选标准**：
- 影响力（30%）
- 思想深度（30%）
- 商业成就（20%）
- 公众关注度（20%）

---

### ⚔️ 【辩场】观点碰撞
选取争议性话题，AI模拟正反双方辩论

**辩题示例**：
- AI是否会取代人类工作？
- 中国电动车是否已经领先全球？
- 知识付费的未来在哪里？

---

### 📚 【书院】深度解读
对每位嘉宾进行深度分析

**分析框架**：
```
1. 人生轨迹 → 理解价值观形成
2. 关键决策 → 分析决策逻辑
3. 核心观点 → 提炼思想精华
4. 实用建议 → 提炼行动指南
```

---

## 三、采访流程升级

### 3.1 五段式采访法

| 阶段 | 时长 | 重点 |
|------|------|------|
| 开场 | 5分钟 | 破冰，了解背景 |
| 人生 | 15分钟 | 成长轨迹与价值观 |
| 事业 | 20分钟 | 关键决策与思考 |
| 观点 | 15分钟 | 深度观点碰撞 |
| 思考 | 5分钟 | 总结与建议 |

### 3.2 追问技巧

**适用场景**：
- 回答过于笼统 → 深入细节
- 观点有新意 → 追问原因
- 存在争议 → 追问反对意见
- 涉及数据 → 追问来源
